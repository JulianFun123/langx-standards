# Dragon LangX standard


### Simple texts
```dragon_langx
hello_world (
    Hello world everyone
)
```


### Variables
```dragon_langx
hello_user (
    Hello, %user%
)
```


### Collections
```dragon_langx
hello_messages {
    hello_world (
        Hello world everyone
    )
    hello_user (
        Hello, %user%
    )
}
```
