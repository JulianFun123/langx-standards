# Snake LangX standard


### Simple texts
```snake_langx
hello_world: Hello world everyone
```

### Multi Line
```snake_langx
hello_world: (\n)-
  Hello
  World
-
```


### Variables
```snake_langx
hello_user: Hello, %user%
```


### Collections
```snake_langx
hello_messages
    hello_world: Hello world everyone
    hello_user: Hello, %user%
```
