# PHP LangX standard


### Simple texts
```php
[
    "hello_world"=>"Hello world everyone"
]
```


### Variables
```php
[
  "hello_user"=>"Hello, %user%"
]
```


### Collections
```php
[
  "hello_messages"=>[
      "hello_world"=>"Hello world everyone",
      "hello_user"=>"Hello, %user%"
  ]
]
```
