# XML LangX standard


### Simple texts
```xml
<hello_world>Hello world everyone</hello_world>
```


### Variables
```xml
<hello_user>Hello, %user%</hello_user>
```


### Collections
```xml
<hello_messages>
    <hello_world>Hello world everyone</hello_world>
    <hello_user>Hello, %user%</hello_user>
</hello_messages>
```
